# JSON-RPC Reference

`anvil-zksync` exposes several JSON-RPC namespaces. Click a namespace to view available methods:

| Namespace                                          | Description                       |
| -------------------------------------------------- | --------------------------------- |
| [`eth_*`](/rpc/eth)                                | Ethereum compatible base methods  |
| [`zks_*`](/rpc/zks)                                | ZKsync specific extensions        |
| [`anvil_*`](/rpc/anvil)                            | Anvil node controls               |
| [`hardhat_*`](/rpc/hardhat)                        | Hardhat style testing helpers     |
| [`evm_*`, `net_*`, `web3_*`, `debug_*`](/rpc/misc) | Miscellaneous utilities & tracing |
